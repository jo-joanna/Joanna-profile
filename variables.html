Task1:

This task covers console.log, alert and document.write. It will show declared variable. I wrote above "document.write" in the page. There will pop up an alert. To see a console log use F12 key.

Task2:

Declare variable numberOfKittens in quantity of 5, numberOfPuppies in quantity 8.
Count the total of puppies and kittens.
Use document.write to say "Anne has got & total of puppies and kittens."
Concatenate sentence: numberOfKittens belongs to Anne's daughter and numberOfPuppies belongs to Anne's son.

To see the outcome you need to open the console tool F12.

Task3:

2. Calculate the new total.

Round the output to 2 decimal places and display the tip amount in the output as well.

Task4:

Write a simple program to combine a first name and a last name inside a function. Then update the function to accept a first and last name as arguments. The outcome of the code is visible after click on F12 key.

Task5:

Control flow and conditions. Write a program to check "shall I wear a coat?"

The outcome can be checked pressing F12 key.

Task6:

Function and switch.
Create a function named calculator. Add 3 arguments for "number1", "number2" and "operator". Use a switch statement to determine which operator to use. Add the math code for each operator in the case statement to return the value. Console.log a message "{number1} {operator} {number2} = {value}"

The outcome can be checked pressing F12 key.

Task7:

Loop.
Write a loop that outputs the 7 times table from 7x1 = 7 to 7x12 = 84

Also use a loop inside a loop to write all the times tables from 1 to 12. The outcome can be checked pressing F12 key.

Task8:

Arrays
Create an array of your favourite foods.

Print some of them to the screen. Next, loop all favourite foods. The outcome can be checked pressing F12 key.

Task9:

Objects
Create an object to hold information on your favourite recipe.

Then display the properties on screen. Next, create a loop to list all the ingredients and directions. The outcome can be checked pressing F12 key.
